# KELEDON PM Weekly Status

This directory contains weekly status reports for the KELEDON project.

## Purpose
Provides PM visibility into:
- Agent work distribution and progress
- Issue status and health
- Blocking items
- System state assessment

## Structure
Each week creates a new file: `YYYY-MM-DD-weekly-status.md`

## Template
```markdown
# Weekly Status - YYYY-MM-DD

## Agent Activity
- Issues claimed: X
- Issues completed: Y
- Issues in progress: Z

## System Health
- Total open issues: X
- Stalled issues: Y
- Average time to completion: Z days

## Blockers & Decisions
[List any blocking items requiring human intervention]

## Risk Assessment
[Assessment of system health and emerging issues]
```

## Notes
- Agents auto-populate this via milestone reports
- PM Agent uses this for priority arbitration
- Human Product Owner reviews for strategic decisions
